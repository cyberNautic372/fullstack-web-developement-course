Box Model & Common Properties

.element { /* Box Model */ width: 300px; height: 200px; padding: 20px; /* Inside spacing */ margin: 10px; /* Outside spacing */ border: 2px solid #000; box-sizing: border-box; /* Colors & Background */ color: #333; background-color: #f0f0f0; background: linear-gradient(to right, red, blue); /* Text */ font-family: Arial, sans-serif; font-size: 16px; text-align: center; line-height: 1.5; /* Positioning */ display: block; /* block, inline, flex, grid */ position: relative; /* static, relative, absolute, fixed */ float: left; /* left, right, none */ }
üí° Tip: Use box-sizing: border-box to include padding and border in element's total width/height.

Flexbox Layout

/* Container Properties */ .container { display: flex; flex-direction: row; /* row, column, row-reverse */ justify-content: center; /* main axis alignment */ align-items: center; /* cross axis alignment */ flex-wrap: wrap; /* wrap, nowrap, wrap-reverse */ gap: 10px; /* space between items */ } /* Item Properties */ .item { flex: 1; /* grow shrink basis */ order: 2; /* change display order */ align-self: flex-start; /* self alignment */ }

CSS Grid Layout

.container { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: auto 1fr auto; gap: 20px; grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } /* Item Placement */ .item { grid-column: 1 / 3; /* start at line 1, end at line 3 */ grid-row: 2; /* row 2 */ grid-column: span 2; /* span 2 columns */ }

Responsive Design & Media Queries

/* Viewport Meta Tag (HTML) */ <meta name="viewport" content="width=device-width, initial-scale=1.0"> /* Media Queries */ @media screen and (max-width: 768px) { .container { flex-direction: column; } h1 { font-size: 1.5rem; } } /* Common Breakpoints */ /* Mobile: max-width: 480px */ /* Tablet: 481px - 768px */ /* Laptop: 769px - 1024px */ /* Desktop: 1025px+ */
‚ö†Ô∏è Important: Always use mobile-first approach. Start with mobile styles, then add media queries for larger screens.
